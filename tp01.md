# Machine Learning

## ¿Que es Machine Learning?

Machine Learning es una **rama de la inteligencia artificial** en al cual la idea es que las computadoras puedan aprender a partir de datos, y no de programaciones de programadores. En vez de decirle al programa que hacer, le dan muchos ejemplos y encuentra patrones.

---

## 5 Empresas que suan machine learning:

### 1. Uala:

- **Nombre del servicio:**
  Deteccion de fraude financiero

- **Problema que resuelve con ML:**
  Identifica fraude o cuentas robadas

- **Tipo de ML:**
  Supervisado, ya que se lo entrena con ejemplos de transacciones fraudulentas

- **Que tipo de datos necesita para funcionar:**
  - Transacciones
  - Montos
  - Frecuencia de pagos
  - Ubicacion
  - Dispositivo usado

### 2. Amazon:

- **Nombre del servicio:**
  Recomendacion de productos

- **Problema que resuelve con ML:**
  Mostrar productos que capaz quieras comprar

- **Tipo de ML:**
  No supervisado, ya que usa un filtrado colaborativo para encontrar similitudes entre productos y compradores

- **Que tipo de datos necesita para funcionar:**
  - historial de compras
  - productos vistos
  - carrito
  - reseñas
  - comportamiento de navegacion

### 3. Spotify:

- **Nombre del servicio:**
  Recomendacion de musica

- **Problema que resuelve con ML:**
  Descubrir canciones que probablemente te gusten

- **Tipo de ML:**
  Mixto: supervisado + no supervisado
  - clustering para agrupar gustos
  - modelos que predicen que cancion te va a gustar

- **Que tipo de datos necesita para funcionar:**
  - canciones escuchadas
  - playlists
  - skips
  - likes
  - caracteristicas del audio

### 4. Instagram:

- **Nombre del servicio:**
  Feed personalizado

- **Problema que resuelve con ML:**
  Mostrar contenido que mas probablemente te interese

- **Tipo de ML:**
  Supervisado, ya que predice probabilidad de que interactues con un post

- **Que tipo de datos necesita para funcionar:**
  - likes
  - comentarios
  - tiempo mirando un post
  - relaciones sociales
  - historial de interaccion

### 5. Uber:

- **Nombre del servicio:**
  Prediccion de demanda y precios dinamicos

- **Problema que resuelve con ML:**
  Ajustar precios y ubicar conductores segun la demanda

- **Tipo de ML:**
  Supervisado, ya que predice cuantos viajes habra en una zona

- **Que tipo de datos necesita para funcionar:**
  - ubicacion
  - horarios
  - clima
  - historial de viajes
  - eventos en la ciudad
